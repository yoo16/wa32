セレクタ

要素の value プロパティ（属性）を入力 

      $(セレクタ).val(値);

input

全て

HTML上の全ての input の value を「初期値」に書き換える 

      $('input').val('初期値書き換え');

ID

ex) input id="title"


      $('#title').val('タイトル書き換え');

クラス

ex) input class="memo"


      $('.memo').val('メモ書き換え');

階層

親要素 > 子要素


    $('親要素 > 子要素')

ex) div > input


      $('div > input').val('メモ書き換え');

属性（プロパティ）セレクタ


        $('セレクタ[属性名]');
        $('セレクタ[属性名=属性値]');

ex) input[name=price]

input name="price" の要素を全て書き換え 

      $('input[name=price]').val('1000');

フォームセレクタ


      $('セレクタ:type属性]')

ex) input:hidden

input type="hidden" の要素を全て書き換え 

      $('input[type=hidden]').val('3');

フィルタ


        $('li:first')
        $('li:nth-child(数値)')
      

        $('li:first input').val('li の最初');
        $('li:nth-child(3) input').val('li の3つ目');
        $('li:last input').val('li の最後');
