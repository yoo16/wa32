HTML

JS:変数を利用しない場合


    $('#itemList li').removeClass('dummy'); //class から dummy を削除
    $('#itemList li').addClass('item');     //class に item を追加
    $('#itemList li').css('color', 'red');  //css に color="red" を追加

JS:変数を利用する場合


    let target = $('#itemList li');
    target.removeClass('item');
    target.addClass('item');
    target.css('color', 'red');

JS:省略（メソッドチェーン）


    let target = $('#itemList li');
    target.removeClass('item').addClass('item').css('color', 'red');
