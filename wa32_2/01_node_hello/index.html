Node.js基本

バージョン確認


      $ node --version
      v13.6.0

ファイル実行


        $ node ファイル名

hello.js


        "use strict";

        console.log("Hello World!");
        

        $ node hello
        //または
        $ node hello.js

REPL ファイル読み込み

読み込みたいファイルディレクトリに移動 


      $ node
      > .load ファイル名

REPL ファイル保存

保存したいファイルディレクトリに移動 


      $ node
      > .save ファイル名
