fadeIn & fadeOut

JS

fadeIn()

アニメーションで表示


      $(セレクタ).fadeIn(duration, easing, complete);

fadeOut()

アニメーションで非表示


      $(セレクタ).fadeOut(duration, easing, complete);
duration
アニメーションの実行時間（ms:ミリ秒)
easing
アニメーションの加速/減速させるための関数
complete
アニメーション完了後に実行される関数

setTimeout()


      setTimeout(function, delay, param1, param2, ...)
function
delay 時間(ms)後に実行される関数
delay
遅延時間(ms)
param1, param2...
関数の引数

Sample


      //fade in
      let photo = $('#photo');
      function open() {
          photo.fadeIn(3000, "swing",
              function () {
                  setTimeout(close, 1000);
              });
      }

      //fade out
      function close() {
          photo.fadeOut(1500, "swing",
              function () {
                  setTimeout(open, 500);
              });
      }
      open();
