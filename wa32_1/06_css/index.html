CSSの設定（callback)

指定したプロパティを、callback で操作

Message

        $(セレクタ).css(プロパティ, function() {
            ...
        })

HTML

JS


        $('#callback').css('width', function(index, value) {
            let width = parseInt(value);
            return width * 0.5;
        });
