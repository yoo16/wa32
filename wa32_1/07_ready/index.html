ready & load event

Ready Event

いずれも同じ処理だが、ready() は使われなくなっている

function()


      $(function() {
          console.log('$');
      });

ready()


      $(document).ready(function () {
          console.log('ready()');
      })

Loaded Event

window on load

DOMだけでなく、全てのアセット（画像など）が読み込まれた場合に発火
Ready Event より後に呼ばれる 


      $(window).on("load", function(){
          console.log('window load');
      });
