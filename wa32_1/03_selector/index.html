セレクタ

基本

要素の value プロパティ（属性）を入力 

      $(セレクタ).val(値);

HTML要素

HTML上の全ての input の value を「初期値」に書き換える

HTML

JS


      $('input').val('初期値書き換え');

ID

例）id="title" の値を設定する

HTML

JS


      $('#title').val('タイトル書き換え');

Class

JS


      $('.memo').val('メモ');

HTML

階層

親要素 > 子要素

親直下の子要素を選択 

      $('親要素 > 子要素')

div 直下の input 要素全て選択


      div > input

HTML

JS


      $('div > input').val('子要素');

親要素 子孫要素

親以降のすべての要素を選択 

      $('親要素 子孫要素')

div 内の input 要素全て選択


      div input

HTML

JS


      $('div input').val('子孫要素');

属性（プロパティ）セレクタ


        $('セレクタ[属性名]');
        $('セレクタ[属性名=属性値]');

input[name=price]

HTML

input name="price" の要素を全て書き換え

JS


      $('input[name=price]').val(0);

フォームセレクタ


      $('セレクタ:type属性]')

input:hidden

input type="hidden" の要素を全て書き換え

HTML

JS


      $('input[type=hidden]').val('3');
HTMLソースを確認

フィルタ


        $('li:first')
        $('li:eq(数値)')  //0, 1, 2...
        $('li:nth-child(数値)')  //1, 2, 3...
        $('li:last')

HTML

JS


        $('li:first input').val(0);
        //$('li:first-child input').val('0');
        //$('li:nth-child(1) input').val('0');
        //$('li:eq(0) input').val('0');

        $('li:eq(2) input').val(500);
        //$('li:nth-child(3) input').val(500);

        $('li:last input').val(1000);

複数選択

JS


      $('.weapon, #shop input').val('複数');

HTML

コンテンツフィルタ


        $('要素1:has(要素2)')
要素2をもつ、要素1を選択

HTML

JS


        $('div:has(p)').css('border: 1px solid #ff0000;');
