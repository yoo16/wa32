フォーカスイベント

フォーカス = 入力項目をクリックしたときに入力できる状態 


      $(セレクタ).on({
          'blur': handler,
          'focus': handler
      });

blur イベントの登録


      $.each(items, function (index, item) {
          item.prop('isSuccess', false);

          item.on({
              'blur': function () {
                  checkEmptyText(index);
                  if (patterns[index]) {
                      checkFormatText(index);
                  }
              },
          });
      });

フォーカスメソッド

focus()

フォーカスしたタイミングで発生 


        $(セレクタ).focus(handler)

blur()

フォーカスを失ったタイミングで発生 


        $(セレクタ).blur(handler)

blur() でイベントを直接実行

blur() でフォーカスをはずし、登録済の buler イベントを発火 


      let check = function () {
          $.each(items, function(index, item) {
              item.blur();
          })
      }

ハンドラも指定可能


        $(セレクタ).blur(function() {
            console.log('フォーカスがはずれました')
        })
