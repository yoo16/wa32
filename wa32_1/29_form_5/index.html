Cookie

Cookie とは

Cookie はブラウザ上にデータを一時的に保存する仕組みのひとつです。

例えば、ショッピングサイトや SNS などの ログインID やパスワードを入力したときにそのデータをブラウザに記憶し、次回以降ログインで入力する手間が省けます。

セッション（Session）

データの一時記憶の仕組みで「セッション（Session）」がありますが、データをサーバに一時保存する点が異なります。 ログイン状態やショッピングカートのデータを他の画面で引き継く処理でよく利用されます。

セキュリティ的にはセッションの方が比較的安全です。 またセッションを利用するにもユーザやブラウザを識別する Session ID が必要なため Cookie に登録したりします。

JavaScript ネイティブ

jQuery の前に JavaScript ネイティブでの Cookie の利用法は以下の通りです。

Cookie の設定


        document.cookie = "name=Tokyo; expires=365";

Cookie の取得


        let values = document.cookie;

Cookie の削除


        document.cookie = "name=; expires=0";

jQuery ではライブライが必要

jQuery で Cookie を利用するにはライブラリが必要です。

今回は「 jquery.cookie.js」を利用します。

Cookie の設定

キーと値の他に、期限やパスなどを指定できます。


      $.cookie(キー, 値, オプション);

例


      $.cookie('name', 'Tokyo');

もし同じキーが存在する場合はパスを指定します。


      $.cookie(キー, 値, { path: '/' })

期限は日数で設定します。


      $.cookie(キー, 値, { expires: 365 })
expires 日数（期限）の設定。 省略時はブラウザ終了まで有効
path パスの設定。サイト全体で使う場合は「/」を設定
domain ドメインの設定。デフォルトは現在利用のドメイン
secure https プロトコルを要求

Cookie の取得

キーを指定して値を取得します。


      $.cookie(キー)

例


      let name = $.cookie('name');

Cookie を全取得

クッキーを全取得し、オブジェクトを返します。


      $.cookie()

例


      let values = $.cookie();
      let name = values.name;

Cookie の削除

Cookie を削除します。


      $.cookie(キー)

もし同じキーが存在する場合はパスを指定します。


      $.cookie(キー, { path: '/' })

サンプル

まず、 Form4 を修正します。

「jquery.cookie.js」をダウンロードするか CDN で読み込みましょう。

HTML

「記憶する」チェックボックスを追加します。


      <p>
        <label for="isSaveCookie">
        <input type="checkbox" id="isSaveCookie">
        記憶する
        </label>
      </p>

初期値


      //Cookie 保存日数
      let sessionExpires = 7;

submit ボタンイベントの修正

「記憶する」にチェックした場合に Cookie 保存、そうでない場合は削除します。 


      target.on('submit', function () {
          if ($('#isSaveCookie').prop('checked')) {
              setCookie();
          } else {
              removeCookie();
          }
          check();
          return false;
      });

入力チェックを外部関数にする

blur に記載した入力チェック処理を外部関数にします。 


      let checkInput = function (index) {
          checkEmptyText(index);
          if (patterns[index]) {
              checkFormatText(index);
          }
          check();
      }
      

      let init = function () {
          ...

          $.each(items, function (index, item) {
              item.prop('isSuccess', false);
              item.on({
                  'blur': function () {
                      //ここを外部関数にした
                      checkInput(index);
                  },
              });
          });

          ...

Cookie設定

データの他に「記憶する」チェックも保存して、次回ページ読み込み時に利用します。 


      function setCookie() {
          $.cookie('isSaveCookie', true, { expires: sessionExpires });
          $.cookie('name', items[0].val(), { expires: sessionExpires });
          $.cookie('furigana', items[1].val(), { expires: sessionExpires });
      }

Cookie取得


      function getCookie() {
          // Cookie全取得
          let cookie = $.cookie();
          if (cookie) {
              // 記憶するチェック
              $('#isSaveCookie').prop('checked', cookie.isSaveCookie);

              items[0].val(cookie.name);
              items[1].val(cookie.furigana);

              // 入力チェック
              $.each(items, function (index, item) {
                  checkInput(index);
              });
          }
      }
