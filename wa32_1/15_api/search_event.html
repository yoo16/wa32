コールバック（callback）

コールバック（callback）は関数を変数定義し、他の関数で実行できるようにする仕組みです。

関数を変数定義すると構造的で複雑な条件でも関数が実行しやすくなり、非同期処理でよく利用されます。

if 文で条件分岐しないで、変数を指定して関数を実行する例です。 


        let methods = {
          hello: {
              callback: function (name) {
                  $('#result').text('callback:' + name)
              }
          },
          bye: {
              callback: function (name) {
                  $('#result').text('callback:' + name)
              }
          },
      };
      
      function main(key) {
          methods[key].callback(key);
      }
      
      main('hello');

イベントサイトのデータ定義

イベントサイトの URL と API が成功したときのコールバック(callback)を定義します。 


      let events = {
          doorkeeper: { 
              url: 'https://api.doorkeeper.jp/events?q=',
              callback: doorkeeper
          },
          connpass: { 
              url: 'https://connpass.com/api/v1/event/?keyword_or=',
              callback: connpass
          }
      }

リクエストの準備

ラジオボタンを取得してイベントの種類を取得し、検索キーワードから API の URL を生成します。 


      let eventKey = $('input[name=event]:checked').val();
      let event = events[eventKey];
      let url = event.url;
      url += $('#word').val();
  
      eventList.text('データを検索中...');

API 接続中は、結果欄に「データを検索中...」と表示するようにしています。

イベントごとのコールバック関数

イベントごとのコールバック関数を実装します。 JSON データの構造が違うため、それにあわせた HTML を作成します。

Doorkeeper


      function doorkeeper(values) {
          $.each(values, function (key, value) {
              let li = $('<li></li>').addClass('list-group-item');
              let a = $('<a></a>',
                  {
                      href: value.event.public_url,
                      text: value.event.title,
                      target: '_blank'
                  });
              li.append(a);
              eventList.append(li);
          });
      }

Connpass


      function connpass(values) {
          $.each(values.events, function (index, value) {
              let li = $('<li></li>').addClass('list-group-item');
              let a = $('<a></a>',
                  {
                      href: value.event_url,
                      text: value.title,
                      target: '_blank'
                  });
              li.append(a);
              eventList.append(li);
          });
      }

JSON型でリクエスト（dataType = json）

dataType を json にすると、foods.json データを JavaScriptオブジェクトに変換して取得できます。 


      $(function () {
          $('#getJson').on('click', function () {
              $.ajax({
                  url: apiUrl,
                  type: 'get',
                  dataType: 'json',
              }).done(function(data) {
                  console.log(data);
                  $('#type').text('json');
                  let text = JSON.stringify(data);
                  $('#result').text(text);
              }).fail(function() {
                  alert('API Error');
              });
          });
      });
