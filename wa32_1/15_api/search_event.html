イベントサイトのデータ定義

イベントサイトの URL と API が成功したときのコールバック(callback)を定義します。 


      let events = {
          doorkeeper: { 
              url: 'https://api.doorkeeper.jp/events?q=',
              callback: doorkeeper
          },
          connpass: { 
              url: 'https://connpass.com/api/v1/event/?keyword_or=',
              callback: connpass
          }
      }

コールバック（callback）とは

コールバック（callback）は、関数を変数に定義しておき、呼び出し先で実行できるようにする仕組みで、非同期処理でよく利用されます。

関数を変数定義しておくことで、複雑な条件でも関数が実行しやすくなります。 


      let functions = {
          hello: { 
              callback: function(name) { alert('Hello ' + name) } 
          },
          bye: {
              callback: function(name) { alert('Bye ' + name) }
          },
      };
      let key = 'hello';
      functions[key].callback('Tokyo');

JSON を利用する

JavaScript で JSON データを利用するには JSONクラスが便利です。

JSON を JS オブジェクトに変換

JSON.parse() で JSON を JavaScriptオブジェクトに変換します。 


        JSON.parse(JSONデータ)

JS オブジェクトを JSON に変換

JSON.stringfy() で JavaScriptオブジェクトを JSON に変換します。 


        JSON.stringfy(オブジェクト)

location から URL を生成する

location を使って、現在の URL から API の URL を生成してみます。

現在の URL


        location.href

現在の URL から index.html を除いた js/foods.json の URL は以下のようになります。

実行前


        http://localhost/samples/index.html

実行後


        http://localhost/samples/js/foods.json

サンプル


      const href = location.href;
      const index = href.lastIndexOf('/') + 1;
      const filename = href.substr(index);
      const baseUrl = href.substring(0, index);
      const apiUrl = baseUrl + 'js/foods.json'

Ajax でリクエスト

GET で 生成した apiUrl にリクエストし、成功は done()、エラーは fail() とそれぞれ処理をします。

テキスト型でリクエスト（dataType = text）

dataType を text にすると、foods.json データを文字列で取得できます。 


      $(function () {
          $('#getText').on('click', function () {
              $.ajax({
                  url: apiUrl,
                  type: 'get',
                  dataType: 'text',
              }).done(function(text) {
                  $('#type').text('text');
                  $('#result').text(text);
                  let data = JSON.parse(text);
                  console.log(data);
              }).fail(function() {
                  alert('API Error');
              });
          });
      });

JSON型でリクエスト（dataType = json）

dataType を json にすると、foods.json データを JavaScriptオブジェクトに変換して取得できます。 


      $(function () {
          $('#getJson').on('click', function () {
              $.ajax({
                  url: apiUrl,
                  type: 'get',
                  dataType: 'json',
              }).done(function(data) {
                  console.log(data);
                  $('#type').text('json');
                  let text = JSON.stringify(data);
                  $('#result').text(text);
              }).fail(function() {
                  alert('API Error');
              });
          });
      });
