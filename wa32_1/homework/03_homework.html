HTML操作

内容

HTML

input id = character_name を作成
ul id = item_list を作成
ul に li を5つ追加し、li に div を追加

      <input type="text" name="character_name">
      <ul id="item_list">
        <li>
          <div></div>
        </li>
        ...
      </ul>

divの中身

div に 以下の要素を追加

p class = item_name
input name = price
input name = amount

      <div>
        <p class="item_name"></p>
        <input type="text" name="price">
        <input type="text" name="amount">
      </div>

Javascript

character_name の要素の値を「HAL」 に設定
item_name の要素のテキストをすべて「アイテム名」を設定
price, amount の要素の値をすべて 「0」 に設定
「アイテム名（item_name）」を変更
  • 最初:「銅のつるぎ
  • 3番目：「皮のたて」
  • 最後：「皮のぼうし」
「価格（price）」を変更
  • 「銅のつるぎ」:150
  • 「皮のたて」：90
  • 「皮のぼうし」：50
・「量（amount）」を変更
  • 要素の値をすべて 「1」 に変更
タイトルとして「武器屋」の文字の入ったHTMLタグ（h2,pなど）を追加
メッセージ欄を作り、「character_name」で設定した名前を利用して、メッセージを表示
例）HALさん、何のご用でしょうか？
